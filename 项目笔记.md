### express生成器 自动生成 mvc代码
```
安装
npm i express-generator -g
在项目目录中打开命令窗口
执行
express -e （-e生成ejs模板引擎）
```

### 隐藏域

```html
<input type="hidden" name="xxx" value="xxx"/>
隐藏域 表单控件类型
作用：
	提交数据（form）时，有一些数据 希望不被用户看见（提交订单时成本价，编辑时id）
就可以使用隐藏域携带这个数据
```
